Titan Panel addon to display Pet Status (based on Titan Pet State by Noctornus)

Button displays (each one can be toggled on/off):
	Pet Name
	Pet Level
	Pet XP (percent)
	Happiness
	Is pet feeding

Tooltip displays:
	Pet Name
	Pet Level
	Happiness
	Experience Points
	Food Types
